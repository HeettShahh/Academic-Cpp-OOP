### 0. Lecture Basics & Syntax
* **Focus:** Language Fundamentals
* **Description:** A collection of introductory scripts written during initial lectures to explore C++ syntax mechanics.
* **Key Concepts:**
    * **Core Syntax:** Standard Input/Output (`cin`, `cout`) and variable initialization.
    * **Logic Control:** Implementation of branching logic (`if`, `else`) and arithmetic operators.
    * **Functions:** Basic function definition and **Function Overloading** (defining multiple functions with the same name but different parameters).
